<h2 align="center">🐾 Everything Your Best Friend Deserves 🐾</h2>

<p align="center">
<b>PawMart</b> (DogCart) is a delightful, full-stack pet e-commerce platform where pet lovers can browse, shop, and checkout premium products for their furry companions. Built with a production-grade tech stack including React, Redux, MUI, and a robust Express + MongoDB backend.
</p>

<img src="https://user-images.githubusercontent.com/73097560/115834477-dbab4500-a447-11eb-908a-139a6edaec5c.gif" alt="divider"/>

```

│   ├── demo-images/                # Product demo screenshots
```

## 🚀 Quick Start

### Prerequisites

```bash
Node.js 16+  •  MongoDB  •  npm or yarn
```

### Installation

```bash
# Clone the repository
git clone https://github.com/Diganta18-noob/DogCart.git
cd DogCart

# ── Backend Setup ──
cd backend
npm install
node seedUsers.js              # Seed user data
npm run dev                    # Starts on http://localhost:8080

# ── Frontend Setup ──
cd ../frontend
npm install
npm start                      # Starts on http://localhost:3000
```

### 🧪 Running Tests

```bash
cd frontend
npm test                       # Run test suite
```
